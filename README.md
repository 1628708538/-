openId获取↓

复制到微信，随便发送给一个人

第一：打开这个debugmm.qq.com/?forcex5=true

第二:然后在打开这个链接http://debugx5.qq.com/

第三：点开《信息》→✓上《打开vConsole调试功能》的勾勾

第四：打开智慧校园，看看右下角有没有《vConsole》

第五：进入智慧校园后点击《vConsole》→《Storsge》→《LocalStorsge》→复制《xd_openId》后面的东西，那个就是openId了
